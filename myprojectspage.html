Project Description

The Safe Night device is a security system that communicates with a companion Android app using Bluetooth Serial communication. It is equipped with an LCD display, a temperature sensor, and a servo motor. As my first Arduino project, it was designed to secure a box or a small container.

This project started during the summer of 2015 after I had completed the beginner starter projects for the Arduino Uno. I was inspired to create this when I saw my old grade 6 science project in the closet, a plastic box which triggered an alarm when opened. I sought to build upon that concept using my newfound skills in hardware and programming.

Key Learnings:
  • Programming Arduino side Bluetooth Serial Communication with HC-05 Bluetooth Module
  • Working with a Liquid Crystal Display to scroll text and continously display status updates
  • Implementing a more accurate analog to digital calculation for the temperature reading
Easy Blind - Pending

Easy Blind is an Arduino device that will utilize a photoresistor to detect the amount of light outside, and adjust the angle of your blinds accordingly. A sensor feedback algorithm will be developed to control the bipolar stepper motor in order to control the blinds with precision

Key Learnings:
  • Understanding control theory and applying the feedback loop in adjusting the blinds
  • Learning how to work with and troubleshoot bipolar stepper motors
  • Using Solidworks to design a piece that will attach the device to the blinds and 3-D printing it
Project Description

This project consists of a line following car powered by Hydrogen PEM Fuel Cells. The car utilized light and colour sensors to navigate a course, and a custom written algorithm written in C++ to handle forks in the path and dead-ends to reach the desired destination. A TI MSP430 microcontroller was used to control the two brushed DC motors.

The most important lesson I learned from this project was that when developing a product, factors besides the hardware and software development must be considered, like the economic, business, and environmental factors. In order to ensure that the car would have the longest run time possible, the fuel cells had to be frequently hydrated. The temperature, humidity were some factors that had to be considered for storage as well. Consequently, this project inspired me to contribute to the eventual goal of vehicles on the road being powered by alternative means of energy

Key Learnings:
  • Learned how to develop a line following algorithm in C++ that could forks and dead-ends
  • Utilized our knowledge of the electrochemical reaction to design our car with double the hydrogen capacity to match the reactant proportions to increase the output voltage.
  • Implemented a calibration function to adjust the sensor according to lighting conditions
  • Equipped the car with sensor debouncing to reduce noise from the colour and light sensors.
Project Description

This robot was built with the LEGO NXT Robot Kit, and programmed using RobotC. It featured a colour sensor that detected the colour, and dispensed a certain amount of seasoning according to the colour detected. An ultrasonic sensor was used to ensure seasoning occured while the path was clear. Two motors with encoders were used to track precise rotations in order to regulate the amount of seasoning dispensed.

The idea behind this project was to automate the process of seasoning dishes immediately before being served to customers in a restaraunt. Plates would be color coded to indicate the desired spice level, and the robot would leverage the data from the color sensor to detect it, and dispense the spice accordingly. Aside from the technical aspect of this project, I also made design decisions using metrics and was responsible for the technical report writing of this project.

Key Learnings:
  • Measuring and machining the seasoning cup holders using bandsaws, and sanders
  • Coding and debugging RobotC code
  • Technical report writing, and utilized metrics to make mechanical design decisions

  • Machined using bandsaws to detect spice level

  • Spice container holders that rotated to dispense certain amounts of seasoning

  • Side view showcasing the colour sensor used to scan plates to indicate spice level

  • Central assembly of robot
